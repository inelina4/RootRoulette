RootRoulette ir interaktīva Python spēle, kurā lietotājam tiek dots vārds angļu valodā un jāmin tā izcelsme (piemēram, grieķu, latīņu vai franču valoda).

Pēc katra mēģinājuma programma parāda pareizo atbildi un paskaidrojumu, saglabā rezultātu un punktu skaitu.
Sākot jaunu spēli, lietotājam ir iespēja norādīt, uz cik jautājumiem viņš vēlās atbildēt.

Šī spēle apvieno programmēšanu ar valodniecību, ļaujot izglītojošā veidā iepazīt angļu valodas saknes.
RootRoulette būtu piemērota aktivitāte angļu valodas leksikoloģijas lekcijās, kad māca par etimoloģiju.
